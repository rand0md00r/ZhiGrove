# 🔄 ZhiGrove 工作流可视化

## 📊 整体流程图

```
📥 收件箱 (00-inbox)
    ↓ 每日整理
💡 想法记录 (30-ideas)
    ↓ 每周整理
📄 论文笔记 (20-papers)
    ↓ 每月整理
🔬 实验记录 (40-experiments)
    ↓ 持续沉淀
📚 知识库 (10-knowledge)
    ↓ 定期总结
📊 报告 (50-reports)
```

## 🕐 时间轴视图

### 每日循环 (5-10分钟)
```
🌅 早晨 (2分钟)     🌙 睡前 (3-5分钟)
     ↓                      ↓
查看收件箱 ←─────────→ 记录新知识
标记重要事项 ←───────→ 整理收件箱
更新学习目标 ←───────→ 快速分类
```

### 每周循环 (30-60分钟)
```
📅 周日整理时间
     ↓
🗂️ 收件箱清理 (15分钟)
     ↓
🔗 内容关联 (20分钟)
     ↓
📚 知识更新 (15分钟)
     ↓
📋 下周计划 (10分钟)
```

### 每月循环 (2-3小时)
```
📈 月末深度整理
     ↓
🏗️ 知识体系梳理 (1小时)
     ↓
✅ 内容质量检查 (1小时)
     ↓
🛠️ 工具流程优化 (30分钟)
     ↓
📊 月度报告 (30分钟)
```

## 🎨 状态指示器

### 🟢 健康状态
- 收件箱 < 10个文件
- 每周有新内容
- 知识库持续增长

### 🟡 注意状态
- 收件箱 10-15个文件
- 本周无新内容
- 知识库增长缓慢

### 🔴 警告状态
- 收件箱 > 15个文件
- 两周无新内容
- 知识库无增长
