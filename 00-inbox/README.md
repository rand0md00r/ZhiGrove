# 📥 收件箱 - 快速记录指南

> **原则：先记录，后整理，不纠结格式**

## 🚀 快速记录方式

### 方式1：直接创建文件
```bash
# 快速创建想法
scripts/new.sh idea "我的新想法"

# 快速创建实验记录
scripts/new.sh experiment "实验名称"

# 快速创建论文笔记
scripts/new.sh paper "论文标题"
```

### 方式2：手动创建（推荐新手）
- 直接在这个目录创建 `.md` 文件
- 文件名格式：`YYYY-MM-DD-简短描述.md`
- 内容随意，想到什么写什么

### 方式3：语音/图片记录
- 语音转文字：用手机录音，稍后整理
- 截图保存：重要信息截图，稍后整理
- 手写笔记：拍照保存，稍后整理

## 📝 记录内容建议

**不要纠结格式，记录这些就够了：**
- 今天学到了什么？
- 有什么新想法？
- 遇到了什么问题？
- 看到了什么有趣的东西？

## 🔄 整理时机

- **每日整理**：睡前5分钟，快速分类
- **每周整理**：周日，深度整理和关联
- **每月整理**：月末，知识体系梳理

## 💡 小贴士

1. **记录比完美重要** - 先记下来，格式可以后续调整
2. **关键词很重要** - 用几个关键词标记，方便后续搜索
3. **关联很重要** - 记录时想想和之前什么内容相关
4. **定期清理** - 这个目录不要积累太多，及时整理

---
*记住：收件箱是临时存放的地方，不是最终归宿！*
