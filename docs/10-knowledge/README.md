# 📚 知识沉淀区

> **这里存放经过整理和验证的领域知识**

## 🎯 什么内容放这里？

- **领域知识手册** - 某个技术领域的系统性知识
- **最佳实践总结** - 经过验证的方法和经验
- **技术架构图** - 系统设计和架构文档
- **术语词典** - 专业术语和概念解释
- **学习路径** - 某个领域的完整学习指南

## 📝 整理标准

### 内容要求
- ✅ 经过验证和测试
- ✅ 有实际应用案例
- ✅ 结构清晰，易于查找
- ✅ 包含必要的代码示例

### 格式要求
- 使用统一的模板
- 包含完整的Frontmatter
- 添加相关标签和链接
- 定期更新和维护

## 🔄 整理流程

1. **收集** → 从收件箱收集相关信息
2. **整理** → 按照逻辑结构组织内容
3. **验证** → 检查内容的准确性和完整性
4. **沉淀** → 放入知识库，建立关联
5. **维护** → 定期更新和优化

## 💡 快速操作

```bash
# 创建新的知识文档
scripts/new.sh knowledge "领域名称"

# 更新知识索引
python scripts/build_index.py
```

## 🏷️ 标签体系

- `llm` - 大语言模型相关
- `moe` - 混合专家模型
- `flow-matching` - 流匹配
- `multimodal` - 多模态
- `robotics` - 机器人学
- `rl` - 强化学习
- `function-call` - 函数调用
- `diffusion` - 扩散模型
