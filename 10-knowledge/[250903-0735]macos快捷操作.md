## Raw Notes

太棒了！给你一份面向“纯键盘流”的 macOS（MacBook Air M4）快捷键速查表。尽量覆盖：全局切换、窗口/桌面管理、文本导航、截图、终端、VS Code/Cursor、主流浏览器。建议先在「系统设置 → 键盘」里开启“键盘导航（Full Keyboard Access）”。

# 必备系统设置

* 开启全键盘访问：系统设置 → 键盘 → 键盘导航 → 开启（用 Tab/箭头键选菜单与对话框控件）
* 调整修饰键：系统设置 → 键盘 → 键盘快捷键 → 修饰键（建议把 Caps Lock 映射为 ⌃Control）
* 触控板角落呼出操作（可选）：系统设置 → 桌面与程序坞 → 热角（配合键盘更顺手）

---

# 全局与应用切换

| 功能            | 快捷键                               |
| ------------- | --------------------------------- |
| 应用切换          | ⌘Tab（按住 ⌘ 再点 Tab 循环），反向：⌘⇧Tab     |
| 在当前应用的多个窗口间切换 | ⌘`（数字 1 左边的键），反向：⌘⇧`              |
| 隐藏当前应用        | ⌘H                                |
| 隐藏其他应用        | ⌥⌘H                               |
| 退出应用          | ⌘Q                                |
| 打开 Spotlight  | ⌘Space（或 ⌃Space，依设置）              |
| Siri/表情与符号    | 地球/🌐fn + Space（或 ⌃⌘Space 调出符号面板） |

---

# 窗口与多桌面（Spaces）

| 功能                  | 快捷键                        |
| ------------------- | -------------------------- |
| 关闭窗口/标签             | ⌘W（多数应用对标签同样有效）            |
| 最小化窗口               | ⌘M                         |
| 全屏/退出全屏             | ⌃⌘F                        |
| Mission Control（总览） | ⌃↑（或 F3）                   |
| 应用窗口总览（App Exposé）  | ⌃↓                         |
| 显示桌面                | F11（或使用热角）                 |
| 切换桌面（Space）         | ⌃→ / ⌃←                    |
| 将窗口拖到其他桌面           | Mission Control 中把窗口拖上方的桌面 |

> **磁贴/分屏**：原生无统一快捷键；可用第三方（见文末）映射“左半屏/右半屏/最大化”等。

| 新建标签       | ⌘T                         |
| 关闭标签       | ⌘W                         |
| 还原关闭的标签    | ⌘⇧T                        |

# 文本编辑与光标移动（系统级 & Emacs 风格也适用于终端）

| 功能               | 快捷键                       |
| ---------------- | ------------------------- |
| 行首/行末            | ⌘← / ⌘→（很多编辑器也支持 ⌃A / ⌃E） |
| 上/下/左/右移字符       | 方向键（也可 ⌃P/⌃N/⌃B/⌃F）       |
| 按词移动             | ⌥← / ⌥→                   |
| 顶部/底部            | ⌘↑ / ⌘↓                   |
| 删除光标前一词          | ⌥⌫（Option+Delete）         |
| 向前删除（Delete 键行为） | fn+⌫                      |
| 删除到行尾            | ⌃K（终端/多数文本框有效）            |
| 撤销/重做            | ⌘Z / ⌘⇧Z                  |
| 粘贴并匹配样式          | ⌥⇧⌘V（部分应用支持）              |

# 终端（Terminal/iTerm2/zsh/bash）

| 功能                    | 快捷键                  |
| --------------------- | -------------------- |
| 中断当前进程                | ⌃C                   |
| 发送 EOF / 退出输入         | ⌃D                   |
| 将作业挂起到后台              | ⌃Z（前台恢复：`fg`）        |
| 清屏                    | ⌃L                   |
| 历史搜索                  | ⌃R（反向增量搜索）           |
| 光标到行首/行末              | ⌃A / ⌃E              |
| 删除到行首/行尾              | ⌃U / ⌃K              |
| 删除前一个单词               | ⌃W（或 ⌥⌫）             |
| 按词移动                  | ⌥B / ⌥F              |
| 打开新标签/分屏（iTerm2 可自定义） | ⌘T / ⌘D（竖分）/ ⌘⇧D（横分） |

> iTerm2 推荐在 **Preferences → Profiles → Keys** 自定义分屏/切窗/切标签等为 ⌘⌥←/→、⌘⌥1…9。

# 提升窗口管理效率（可选第三方）

* **Rectangle**（免费）：给窗口平铺/半屏/三分之二等映射成快捷键。常用：⌥⌘←/→（左/右半屏）、⌥⌘↑/↓（上/下半屏）、⌥⌘Enter（最大化）。
* **Hammerspoon + yabai/skhd（进阶）**：完全键盘化平铺与脚本化管理。
* **Raycast/Alfred**：全局启动器 + 跨应用动作（键盘为王）。

---

# 一次性配置清单（建议）

1. 键盘导航：开启；Caps → Control
2. Spotlight：保留 ⌘Space；给 Raycast（如有）设置 ⌥Space
3. Rectangle：启用半屏/三分屏/最大化的快捷键
4. VS Code/Cursor：把“在编辑器组间切换”“切换标签”“多光标”等设置成你最顺手的组合
5. iTerm2：把“分屏/切屏/切标签”映射为 ⌘⌥方向键 + ⌘数字
